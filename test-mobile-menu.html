Teste Menu Mobile

Demonstração do menu hamburguer responsivo

Como testar:

  1. Redimensione a janela para menos de 1024px de largura
  2. Clique no botão hamburguer (☰) para abrir o menu
  3. Clique fora do menu (na área escura) para fechá-lo
  4. Ou pressione a tecla ESC para fechar
  5. O menu fecha automaticamente ao clicar em um link

Funcionalidades implementadas:

  • ✅ Botão hamburguer animado (transforma em X)
  • ✅ Sidebar deslizante da esquerda
  • ✅ Overlay escuro para fechar clicando fora
  • ✅ Fechar com tecla ESC
  • ✅ Fechar automaticamente ao clicar em links
  • ✅ Responsive design
  • ✅ Transições suaves
